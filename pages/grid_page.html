How did I do this?

I used a grid!

A CSS grid is great for designating layouts in two dimensions (both rows and columns.)

You can also use units such as fr to improve responsiveness of objects when screen sizes change. fr represents a fraction of the available space in the grid container, so the objects will change dimensions along with the screen size.

You can also use media queries to set more specific layouts for specific ranges of screen dimensions. As you can see on this page, at a certain screen dimension, the number of columns goes from three to two to one!
