Here, we are simulating how a dozen cupcakes might be placed in container boxes of various sizes. The cupcakes are always the same size and must maintain space between each other, but how their arrangements in the shipping box change depending on the shape of the box.

How did I do this?

I used flexbox!

Flexbox is great for designating distribution across a single axis, and allows you to set constraints regarding if elements shrink, grow, or wrap when the elements' container changes in size.

You can use flex: grow shrink basis to set how items will expand if extra space in container, how items will shrink it not enough space in container and the initial (or "ideal") size of flex items are.
